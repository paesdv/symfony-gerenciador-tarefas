Tecnologias Utilizadas

    Backend:

        PHP 8.2+

        Symfony 7

        Doctrine ORM

    Banco de Dados:

        SQLite (leve e sem necessidade de instalação de um servidor)

    Frontend:

        Twig (Template Engine do Symfony)

        Bootstrap 5 (para um design limpo e responsivo)

        Webpack Encore (para compilação dos assets)
